# InspectnGo Landing Page

React + TypeScript landing page built with Vite, Tailwind CSS, and shadcn/ui.

## Prerequisites

- [Bun](https://bun.sh/) (package manager & runtime)
- Node.js 18+

## Install

```sh
bun install
```

## Development

```sh
bun run dev
```

Opens a local dev server (default: http://localhost:5173).

## Build

```sh
bun run build
```

Output goes to `dist/`.

## Preview production build

```sh
bun run preview
```

## Lint

```sh
bun run lint
```

## Test

```sh
bun run test          # single run
bun run test:watch    # watch mode
```
